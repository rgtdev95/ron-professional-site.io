# Ron Tactay - Portfolio Site

My personal portfolio website showcasing my work in DevOps, cloud infrastructure, and systems engineering.

## What's in here

- `ron-site.html` - Main portfolio page
- `icons/` - Technology icons (Azure, Docker, Kubernetes, etc.)

- HTML + TailwindCSS
- Lucide icons
- Space Grotesk / Space Mono fonts

## Run locally

Just open `ron-site.html` in your browser.
